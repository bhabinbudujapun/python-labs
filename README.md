# Python Labs

My Python fundamentals practice, following the
[algomaster.io Python roadmap](https://algomaster.io/learn/python/course-roadmap),
on the path to learning Django.

Each folder holds exercises for a topic (e.g. `classes/`, `oop basics/`),
and new topics are added as I work through the roadmap.

## Getting started

Clone the repository:
```bash
git clone https://github.com/bhabinbudujapun/python-labs.git
cd python-labs
```

Requires [Python 3](https://www.python.org/downloads/) (check with `python --version`).

## Running an exercise

Run any exercise file by passing its path to `python`:
```bash
python <path/to/file.py>
```

For example:
```bash
python "oop basics/03_exercise.py"
python number_guessing_game.py
```
